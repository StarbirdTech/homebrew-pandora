# Starbirdtech Pandora

## How do I install these formulae?

`brew install starbirdtech/pandora/<formula>`

Or `brew tap starbirdtech/pandora` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "starbirdtech/pandora"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
